Sacolas Abandonadas

Gerencie e acompanhe os contatos com clientes que abandonaram suas sacolas

Como funciona o módulo de Sacolas Abandonadas?

Este módulo exibe clientes que adicionaram produtos ao carrinho mas não finalizaram a compra. Você pode entrar em contato via WhatsApp para recuperar estas vendas.


Como usar:

  • Na aba Para Contatar, você verá todos os clientes com sacolas ativas ainda não contatados.
  • Clique no botão WhatsApp ou Compartilhar Sacola para enviar uma mensagem.
  • Após o envio, confirme se o contato foi bem-sucedido — o sistema registra automaticamente.
  • A aba Histórico mostra todos os contatos realizados, do mais recente para o mais antigo.
